# Price Calculator

A mobile-friendly calculator for calculating product prices by weight.

## Features
- Add custom products with prices
- Calculate cost by weight in grams/kg
- Data saved locally in browser
- Responsive design for mobile

## How to Use
1. Add products with their price per kg
2. Select product and enter weight
3. Get instant cost calculation
